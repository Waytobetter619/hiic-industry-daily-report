# HIIC 产业日报订阅系统

深圳20+8产业集群自动化资讯聚合与推送系统，帮助用户高效获取产业动态，助力投资决策、市场研究和竞品监控。

---

## 🎯 技能定位

**为什么需要这个技能？**

产业研究和投资决策需要持续跟踪行业动态，但手动搜索、筛选、整理信息耗时耗力。这个技能通过自动化流程解决这个痛点：
- ✅ **自动化**：双引擎并行搜索，无需手动查找
- ✅ **时效性**：只抓取24小时内新闻，信息新鲜
- ✅ **智能化**：自动去重、提取、生成摘要
- ✅ **推送化**：定时推送，无需主动查询

**适用场景**：
- 产业研究员跟踪行业动态
- 投资者发现投资机会
- 企业管理者监控竞争对手
- 内容创作者获取写作素材
- 学习者了解行业趋势

## 🚀 核心工作流

### 1. 识别需求

**触发条件**：
- 明确说："产业日报"、"订阅人工智能资讯"
- 模糊表达："帮我看看新能源有什么动态"、"追踪一下半导体行业"

**识别产业**：
- 匹配深圳20+8产业集群（见 `references/industries/`）
- 或提取用户自定义关键词

### 2. 生成日报

**执行步骤**：
```bash
# 使用预设产业配置
node {baseDir}/scripts/generate-report-v7.3.mjs \
  --industry "人工智能" \
  --maxResults 30 \
  --saveToBitable true

# 或使用自定义关键词
node {baseDir}/scripts/generate-report-v7.3.mjs \
  --keywords "固态电池,新能源,锂电池" \
  --maxResults 30
```

**工作原理**（了解即可，无需向用户解释）：
1. **双引擎并行搜索**：Tavily API（新闻模式） + Brave Search API（实时搜索）
2. **严格24小时过滤**：验证发布时间，只保留24小时内新闻
3. **智能去重**：基于URL标准化去重
4. **内容提取**：Jina Reader API提取正文（90%成功率）
5. **摘要生成**：生成50-200字符的语义完整摘要
6. **质量过滤**：排除百科、招聘等噪音
7. **飞书存储**：自动存入多维表格（`scripts/feishu-config.json`）
8. **Markdown输出**：生成可读性强的日报格式

### 3. 询问订阅

生成日报后，**总是询问**用户是否需要订阅：

```
📰 日报已生成！包含 [X] 条最新资讯。

您是否希望订阅此日报？
1. 每日推送（每天早上 9:00）
2. 每周推送（每周一早上 9:00）
3. 自定义时间
4. 暂不订阅
```

**为什么总是询问？**
- 日报的价值在于持续跟踪，一次性查看意义有限
- 订阅后用户无需主动查询，信息自动推送
- 提高用户粘性和技能使用频率

### 4. 设置订阅

**如果用户选择订阅**：

```bash
# 每日推送
openclaw cron add "0 9 * * *" \
  "node {baseDir}/scripts/generate-report-v7.3.mjs \
   --industry '人工智能' --push"

# 每周推送
openclaw cron add "0 9 * * 1" \
  "node {baseDir}/scripts/generate-report-v7.3.mjs \
   --industry '人工智能' --days 7 --push"
```

**订阅管理**：
```bash
# 查看订阅
node {baseDir}/scripts/subscription-manager.mjs list

# 取消订阅
node {baseDir}/scripts/subscription-manager.mjs remove <id>
```

## 🔧 技术细节

### 依赖要求

**必需的API密钥**（用户需要配置）：
```bash
export TAVILY_API_KEY="tvly-xxx"      # Tavily搜索API
export BRAVE_API_KEY="xxx"             # Brave搜索API
```

**可选配置**：
```bash
# 代理设置（如需要）

export HTTPS_PROXY="http://127.0.0.1:7899"
```

### 自定义配置系统

**HIIC日报支持用户自定义配置**，你可以：

- ✅ 添加自己的RSS数据源
- ✅ 修改产业关键词
- ✅ 添加自定义产业
- ✅ 调整主题映射

**快速开始**：

```bash
# 1. 创建工作区配置
mkdir -p workspace/config
cp config/defaults/*.json workspace/config/

# 2. 编辑配置
vim workspace/config/sources.json

# 3. 使用自定义配置
node scripts/generate-report.mjs \
  --industry "人工智能" \
  --config workspace/config
```

**配置文件**：

- `config/defaults/industries.json` - 28个产业配置
- `config/defaults/sources.json` - 数据源配置（可自定义）
- `config/defaults/topics.json` - 主题映射

**详细文档**：见 `references/config-guide.md`

### 性能指标

**数据质量**：
- **时效性**：100% 24小时内新闻
- **覆盖率**：热门行业15-22条/天，小众行业5-10条/天
- **准确性**：90% 内容提取成功率
- **摘要质量**：50-200字符，语义完整

**API配额消耗**：
- Tavily: 6-10次/生成
- Brave: 6-10次/生成
- Jina Reader: 15-25次/生成

**建议使用频率**：
- 每个产业每天最多生成2-3次
- 避免频繁测试消耗配额

## 🐛 故障排查

### 常见问题

**问题1：搜索无结果**
- 检查API密钥：`echo $TAVILY_API_KEY`
- 测试网络连接
- 验证关键词是否过于狭窄

**问题2：推送失败**
- 检查飞书配置：`cat ~/.openclaw/workspace/hiic-push-config.json`
- 验证chatId是否正确
- 确认权限配置

**问题3：摘要质量差**
- 检查内容提取日志
- 验证Jina Reader API状态
- 查看原始网页是否可访问
